Steps to follow to create this project: 
    1.Create a VPC 
    2. Create an Internet Gateway
    3. Create a Custom Route Table
    4. Create a Subnet 
    5. Associate Subnet with Route Table 
    6. Create a Security Group to allow port 22, 80 and 443. 
    7. Create a Network Interface with an IP in the subnet that was created in Step 4.
    8. Assign an Elastic IP to the interface created in Step 7.
    9. Create an Ubuntu Server and Install/Enable Apache2.
